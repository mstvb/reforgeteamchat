# Reforge | TeamChat

## Description

*A Bukkit Plugin for 1.12.2 for TeamChat*

## Features


* TeamChat
    * Login / Logout
    * Notification


## Getting Started


### Installing


* Get Repository

```git clone https://github.com/mstvb/reforgeteamchat.git```


### Commands

```/tc``` > List of Players | Logged In / Logout

```/tc *your Message*``` > Send Message over TeamChat

```/tc login``` > Login

```/tc logout``` > Logout

```/tc list``` > List

```/tc notify``` Notification On / Off
